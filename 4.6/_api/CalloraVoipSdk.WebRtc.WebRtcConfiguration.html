Immutable configuration for a WebRtcClient (the direct-construction surface; the DI/options path adds a mutable counterpart in a later slice). All fields are optional — a zero-config new WebRtcClient() binds an ephemeral loopback endpoint, offers Opus audio, and uses a fresh per-peer DTLS identity.

SimulcastLayers

Send-side simulcast layers to offer (RFC 8853), by a=rid id in send order, e.g. ["hi", "mid", "lo"]. Empty (default) offers a single video stream. When set, the app sends each layer's encoded frames via SendVideoFrameAsync(string, ReadOnlyMemory<byte>, uint, CancellationToken) — the SDK packetises each on its own SSRC with the RID header extension (RFC 8852). Requires EnableVideo. This peer must be the offerer for the simulcast to be advertised.
