SipTransport

Default SIP signaling transport a consumer selects via DefaultTransport. Determines which transport the SDK uses for outbound requests and its advertised local contact when a target URI does not force a transport (for example via a sips: scheme or ;transport= parameter). All five transports are listened on regardless; this only selects the default for outbound routing.
