**Datei:** `docs/audit/CODE_FINDINGS_REGISTER.md`
**Stand:** 2026-07-22

## Zweck und Herkunft

Der Quellcode dieses Repos (`src/`, `tests/`, `perf/`) referenziert in Kommentaren und XML-Dokumentation interne Finding- und Entscheidungs-Marker (`CF-…`, `HARD-…`, `ADR-…`, `CORE-…` sowie die NAT-Marker `N1`/`N2`). Das Original-Register dieser Marker lag **außerhalb des Repos** und ist nicht überliefert. Dieses Dokument ist eine **Rekonstruktion**: Für jeden im Code referenzierten Marker wurden alle Fundstellen gesammelt und die Beschreibung **ausschließlich aus dem umgebenden Code-/Kommentar-Kontext** abgeleitet. Die Beschreibungen geben also den im Code verankerten Stand wieder — nicht zwingend den vollständigen Wortlaut des ursprünglichen Findings.

Konventionen:

- Fundstellen sind repo-relativ als `Datei:Zeile` angegeben; bei vielen Fundstellen die wichtigsten 3–5 plus „u. a.".
- Marker-Familien: **ADR** = Architekturentscheidungen, **CF** = Code-Findings (Protokoll-Korrektheit), **HARD** = Hardening-Findings (Sicherheit, Nebenläufigkeit, Ressourcen, Code-Qualität), **CORE** = Kern-Feature-/True-up-Marker, **N** = NAT-Adressquellen im SIP-Kanal.
