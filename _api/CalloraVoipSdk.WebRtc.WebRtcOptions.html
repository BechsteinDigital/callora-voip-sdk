Remarks

This is the WebRTC facade's own option surface: a sibling of the SIP VoipOptions, not a section nested inside it. A pure-WebRTC host therefore never carries SIP settings, and the two facades meet only at the composition layer (AddCalloraVoip(...).AddWebRtc(...)). See ADR-012.
