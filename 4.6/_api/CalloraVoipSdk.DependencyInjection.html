Namespace CalloraVoipSdk.DependencyInjection

Classes

CalloraBuilder

Builder for optional CalloraVoipSdk dependency overrides.

CalloraHostedService

Coordinates CalloraVoipSdk lifecycle through host start/stop hooks.

CalloraStunServerBuilder

Builder for optional STUN server-host overrides (Level 3). Returned by AddCalloraStunServer(IServiceCollection, Action<StunServerHostOptions>?).

CalloraTurnServerBuilder

Builder for optional TURN/STUN server-host overrides (Level 3). Returned by AddCalloraTurnServer(IServiceCollection, Action<TurnServerHostOptions>?); mirrors CalloraWebRtcBuilder.

CalloraWebRtcBuilder

Builder for optional WebRTC-facade dependency overrides (Level 3). Mirrors CalloraBuilder for the SIP facade and is returned by AddCalloraWebRtc(IServiceCollection, Action<WebRtcOptions>?) and by AddWebRtc(Action<WebRtcOptions>?).

ServiceCollectionExtensions

Dependency-injection entrypoint for CalloraVoipSdk.

StunServerServiceCollectionExtensions

Dependency-injection entrypoint for the STUN server host. Standalone counterpart to AddCalloraTurnServer(IServiceCollection, Action<TurnServerHostOptions>?) for a pure-STUN (Binding) deployment.

TurnServerServiceCollectionExtensions

Dependency-injection entrypoint for the TURN/STUN server host. Standalone counterpart to AddCalloraVoip / AddCalloraWebRtc: a host that runs a TURN server calls AddCalloraTurnServer(IServiceCollection, Action<TurnServerHostOptions>?) and receives a builder for optional overrides.

VoipOptionsValidator

Startup validation for host-bound VoipOptions.

WebRtcServiceCollectionExtensions

Dependency-injection entrypoint for the WebRTC facade. Standalone counterpart to AddCalloraVoip(IServiceCollection, Action<VoipOptions>?): a pure-WebRTC host calls only AddCalloraWebRtc(IServiceCollection, Action<WebRtcOptions>?); a host that wants both facades chains AddCalloraVoip(...).AddWebRtc(...) (ADR-012, two-facade composition).
