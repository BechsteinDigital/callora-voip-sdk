Negotiated media parameters for one active call leg. Produced by the infrastructure adapter after SDP offer/answer exchange and consumed by the application media orchestrator to set up RTP I/O. A record so a derived leg (e.g. after ICE candidate-pair selection) can be produced with a with expression that carries every field across — a hand-written copy had silently dropped the SDES/DTLS key material and video parameters when overriding the endpoints.
