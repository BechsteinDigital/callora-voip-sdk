Commercial modules

Status: in development — not yet available. The pages in this section describe planned commercial plugins and their intended integration surface. They are not shipping products yet. Nothing here is a support or availability commitment.

The SDK core is open and free. Advanced capabilities are planned as paid plugins on a private feed, built on the two public extension points the core already exposes:

  • the module registry (client.Modules, Get<T>/TryGet<T>)
  • the media-tap contract (CreateReceiver/CreateSender)

Because these are the same public contracts your own in-house modules can use, the plugin model adds no private hooks — a commercial module is "just" a module that consumes the tap.
