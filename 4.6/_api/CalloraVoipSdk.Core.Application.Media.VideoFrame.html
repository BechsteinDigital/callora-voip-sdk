One encoded video frame on the public media contract. The SDK is transport-only: the payload carries already-encoded codec bytes (one coded picture) as negotiated for the call — it is not decoded pixels, and the SDK never encodes or decodes. Bring your own codec (VP8, H.264, …) and read PayloadType to identify it.

PayloadType int

Negotiated RTP payload type identifying the video codec.

RtpTimestamp uint

The frame's RTP timestamp on the video clock (90 kHz). All RTP packets carrying this frame share this value; it is absolute, not a duration.

IsKeyFrame bool

true when the frame is (or is known to be) an intra-coded key frame decodable without prior frames; false for a delta frame or when the type is not known.
